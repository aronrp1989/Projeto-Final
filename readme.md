CodeForge by Aron

Portfólio pessoal desenvolvido para apresentar meus projetos, habilidades e informações de contato na área de Desenvolvimento de Software.

Sobre o Projeto

Este site foi criado como parte dos meus estudos em desenvolvimento Front-end.  
O objetivo é reunir em um só lugar meus projetos, competências técnicas e formas de contato, servindo como minha vitrine profissional online.

O layout foi desenvolvido com foco em:

- Visual moderno
- Identidade visual consistente
- Responsividade para desktop e dispositivos móveis
- Navegação simples e intuitiva

---

Tecnologias Utilizadas

- HTML5: Estrutura da página
- CSS3: Estilização, layout responsivo e design moderno
- JavaScript: Interatividade (menu mobile e funcionalidades dinâmicas)

---

Funcionalidades

- Menu responsivo para dispositivos móveis  
- Layout adaptável para desktop, tablet e celular  
- Seções organizadas: Sobre, Projetos, Habilidades e Contato  
- Cards interativos com efeito hover  
- Design com identidade visual em tons de verde tech  
- Links para redes sociais  
- Estrutura pronta para expansão futura

---

Responsividade

O site foi desenvolvido utilizando Media Queries para garantir boa visualização em:

Computadores
Celulares
Tablets

---

Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

- Estruturação semântica com HTML
- Flexbox para alinhamento de layouts
- Organização visual com espaçamento consistente
- Manipulação de classes com JavaScript
- Construção de layouts responsivos
- Padronização de componentes visuais
